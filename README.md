# SkillUPAI
AI-powered Skill Gap Analyzer that compares user skills with job requirements, identifies missing skills, and provides personalized learning recommendations using Spring Boot, React, JWT Authentication, and OpenAI integration.


---

## 🛠️ Tech Stack

### Backend
* **Language:** Java 21
* **Framework:** Spring Boot (v4.0+)
* **Security:** JWT (JSON Web Tokens) & Spring Security
* **Database:** MongoDB

### Frontend
* **Library:** React.js
* **Styling:** Tailwind CSS / CSS3
* **State Management & Routing:** React Router, Axios

### AI & Integration
* **AI Core:** REST API integrations with AI models for dynamic learning path generation and skill analysis.

---

## ✨ Key Features

* **AI-Powered Skill Analysis:** Processes user inputs to accurately pinpoint missing competencies for targeted roles.
* **Dynamic Learning Paths:** Generates tailored, step-by-step roadmaps rather than static, pre-defined guides.
* **Secure Authentication:** Implements stateless JWT authentication to handle secure user registration, login, and session persistence.
* **Responsive Dashboard:** A clean, modern user interface built with React to easily track progress and view generated roadmaps.
* **Robust API Architecture:** Built with clean separation of concerns using Spring Boot's Controller-Service-Repository pattern.
